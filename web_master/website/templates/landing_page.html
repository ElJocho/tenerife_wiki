Dies ist die Website zur großen Geländeübung, welche im März 2022 auf Teneriffa stattfand.

Auf der Karte werden alle auf der Insel gefahrenen Routen angezeigt sowie die Orte, welche im Rahmen der Geländeübung besichtigt wurden. Zusätzlich wird auf der Karte auch identifizierte Vegetation verortet. Die besichtigten Orte werden nach thematischem Inhalt sowie Vegetation nach Vegetationszone unterschiedlich eingefärbt auf der Karte angezeigt. Durch den Slider im linken unteren Karteneck können entweder die Daten aller Tage oder eines einzelnen Tages eingeblendet werden. Durch das Layer-Symbol rechts oben können die drei Datenthemen ein- bzw. ausgeschalten werden. Standardmäßig sind die Routen und besuchten Orte aktiviert, die identifizierte Vegetation jedoch zunächst ausgeblendet.

Sobald sich der Mauszeiger über einem Punkt befindet, so wird ein Übersichtsbild sowie Name des Punktes eingeblendet. Stellt der Punkt identifizierte Vegetation dar, so kann mit einem klick auf den Namen das verknüpfte Pflanzenprofil auf ‚Pl@ntNet‘ aufgerufen werden. Stellt der Punkt einen besichtigten Ort dar, so wird mit einem Klick auf diesen alle Uploads anderer Teilnehmer zu diesem Ort eingeblendet. Außerdem können selbst weitere Bilder und Kommentare hinzugefügt werden. Auf diese Weise entsteht ein interaktiver, gemeinschaftlicher Exkursionsbericht.

Der Quellcode für diese Website findet sich offen unter unserem github.
